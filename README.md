# Metta Holística Web

Este proyecto es una migración desde Wix hacia código puro para reducir costos y mejorar rendimiento.

## Objetivo
Replicar la web original:
https://www.mettaholistica.com/

## Requisitos
- Mantener los mismos colores
- Mantener tipografía lo más similar posible
- Mejorar diseño sin cambiar la identidad visual
- Optimizar velocidad y estructura

## Stack
HTML, CSS, JavaScript (posible migración a framework moderno si es necesario)

## Notas
Este proyecto será trabajado con ayuda de IA (Codex) para desarrollo asistido.
